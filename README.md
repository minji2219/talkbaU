**맡은역할**: 기획 및 디자인 참여, 개발 전 부분 담당

### 주요 기능
- 선물을 받는 사람과 선물을 주는 사람의 **카카오톡 대화를 분석**하여 **선물을 추천해주는 서비스**
- 대화를 첨부하면 대화 내용 중 선물을 받는 사람이 많이 사용하는 단어 10개를 추출하여 ChatGPT와 연결해 선물 5개를 추천 받는다.
- 다시 추천 받기를 이용하여 추천 받았던 선물 외의 선물들을 다시 추천 받을 수 있다.
